# 🛡 Контроль полисов — Форма 2

Локальное веб-приложение для ежемесячного сравнения отчётов по неурегулированным убыткам (лист «Форма 2» из .xls/.xlsx файлов).

## Что делает

- **Исчезнувшие полисы** — были в предыдущем месяце, отсутствуют в текущем
- **Новые полисы** — появились в текущем месяце
- **Расхождения** — изменения в РЗНУ и Доле перестраховщика
- **Отчёт в Excel** — скачивается одной кнопкой

## Установка

### 1. Установить Node.js

Скачай и установи Node.js (версия 18+):  
👉 https://nodejs.org/

Проверь установку:
```bash
node --version
npm --version
```

### 2. Установить зависимости

Открой терминал в папке проекта и выполни:

```bash
cd policy-checker
npm install
```

### 3. Запустить

```bash
npm run dev
```

Приложение откроется в браузере по адресу http://localhost:3000

## Использование

1. Нажми **«+ Загрузить отчёт»**
2. Выбери .xls или .xlsx файл (должен содержать лист «Форма 2»)
3. Введи дату отчёта (например: 01.03.2026)
4. Загрузи второй файл за следующий месяц
5. Сравнение произойдёт автоматически
6. Нажми **«↓ Скачать отчёт в Excel»** для выгрузки

## Структура проекта

```
policy-checker/
├── index.html          # HTML-шаблон
├── package.json        # Зависимости
├── vite.config.js      # Конфигурация Vite
└── src/
    ├── main.jsx        # Точка входа
    ├── styles.css      # Глобальные стили
    ├── App.jsx         # Главный компонент
    ├── parser.js       # Парсинг Excel + сравнение + отчёт

```

## Данные

Все данные хранятся **только локально** в localStorage браузера. Ничего не отправляется на сервер.
